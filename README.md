MiMiProxy (experimental)
========================
Small SOCKS5 proxy server on python3.

### use:
```shell
proxy.py 0.0.0.0 1080
```

### Requirements
python 3.4 or python 3.3 with asyncio module installed

### Roadmap
* setup.py
